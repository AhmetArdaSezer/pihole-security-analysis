# 🛡️ Güvenlik Testi Metodolojisi

Bu projede gerçekleştirilen sızma testi (penetration testing) ve zafiyet analizi adımlarında **OWASP Top 10** (A01: Broken Access Control & A08: Software and Data Integrity Failures) standartları referans alınmıştır. 

Zafiyet tespit süreci, "Cyber Kill Chain" adımlarına uygun olarak bilgi toplama ve sömürü (exploitation) aşamaları şeklinde raporlanmıştır.
